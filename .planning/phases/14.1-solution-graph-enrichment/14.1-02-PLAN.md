```csharp
var realNodes = snapshot.Nodes.Where(n => n.NodeKind == NodeKind.Real).ToList();
```
Use `realNodes` instead of `snapshot.Nodes` wherever nodes are iterated for indexing. Also update `_nodes` dictionary population to use `realNodes` (so `GetAsync` only returns real nodes). Ensure `LoadIndexAsync` also filters if it populates `_nodes` from snapshot.

**InMemorySearchIndex.cs:**
Same pattern — in `IndexAsync`, filter `snapshot.Nodes` to only `NodeKind.Real` before building the in-memory index structures. Ensure the `_nodes` backing store only contains real nodes.

**Important:** Do NOT filter edges — edges referencing stub nodes are still valid for `get_references` queries. Only filter nodes from the search index.
