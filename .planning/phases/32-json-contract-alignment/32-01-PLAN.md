**Step 2: Create `DocCommentConverter` in `src/DocAgent.McpServer/Ingestion/DocCommentConverter.cs`.**

Create a custom `JsonConverter<DocComment>` that handles shape mismatches:
- TS `example: string|null` -> C# `Examples: IReadOnlyList<string>` (wrap single string in list)

- TS `see: string[]` -> C# `SeeAlso: IReadOnlyList<string>` (name mapping)
- TS `params: Record<string,string>` -> C# `Params: IReadOnlyDictionary<string,string>` (compatible with case insensitive)
- TS `typeParams: Record<string,string>` -> C# `TypeParams: IReadOnlyDictionary<string,string>` (compatible)
- `summary`, `remarks`, `returns` map directly (case insensitive handles it)
