**Add DiffAsync to KnowledgeQueryService:**

1. Start Stopwatch
2. Load snapshot A via `_snapshotStore.LoadAsync(a.Id, ct)` — if null, return `QueryResult.Fail(QueryErrorKind.SnapshotMissing, $"Snapshot '{a.Id}' not found")`
3. Load snapshot B via `_snapshotStore.LoadAsync(b.Id, ct)` — if null, same error
4. Build dictionaries: `nodesA = snapshotA.Nodes.ToDictionary(n => n.Id)`, same for B
5. Compute sets: `removedIds = nodesA.Keys.Except(nodesB.Keys).ToHashSet()`, `addedIds = nodesB.Keys.Except(nodesA.Keys).ToHashSet()`, `commonIds = nodesA.Keys.Intersect(nodesB.Keys)`

**Rename detection (per CONTEXT.md — PreviousIds tracking):**
6. For each id in `addedIds.ToList()`:
   - Get `nodesB[id].PreviousIds`
   - Find first `prevId` where `removedIds.Contains(prevId)`
   - If found: emit `DiffEntry(id, Modified, $"renamed from {prevId.Value}")`, remove `prevId` from `removedIds`, remove `id` from `addedIds`

**Remaining changes:**
7. For each remaining `removedId`: emit `DiffEntry(id, Removed, "symbol removed")`
8. For each remaining `addedId`: emit `DiffEntry(id, Added, "symbol added")`
9. For each `commonId`: compare fields and build change summary:
   - `a.DisplayName != b.DisplayName` → `"name: {a} → {b}"`
   - `a.FullyQualifiedName != b.FullyQualifiedName` → `"fqn: {a} → {b}"`
   - `a.Accessibility != b.Accessibility` → `"accessibility: {a} → {b}"`
   - `a.Kind != b.Kind` → `"kind: {a} → {b}"`
   - `a.Docs?.Summary != b.Docs?.Summary` → `"doc changed"`
   - If any changes → emit `DiffEntry(id, Modified, string.Join("; ", changes))`

10. Create `GraphDiff(entries.ToList())`, wrap in ResponseEnvelope (use snapshot B's ContentHash as version), return Ok

**Use snapshot B's ContentHash as the envelope's SnapshotVersion** since the diff describes changes relative to B.
