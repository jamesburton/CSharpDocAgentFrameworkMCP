Create `SnapshotStore` class:

```csharp
public sealed class SnapshotStore
{
    private readonly string _artifactsDir;

    public SnapshotStore(string artifactsDir);

    /// Serialize snapshot, compute content hash, write to artifacts/{hash}.msgpack, update manifest.
    /// Returns the snapshot with ContentHash set.
    public async Task<SymbolGraphSnapshot> SaveAsync(
        SymbolGraphSnapshot snapshot,
        string? gitCommitSha = null,
        CancellationToken ct = default);

    /// Load a snapshot by content hash.
    public async Task<SymbolGraphSnapshot?> LoadAsync(string contentHash, CancellationToken ct = default);

    /// List all snapshots from manifest.
    public async Task<IReadOnlyList<SnapshotManifestEntry>> ListAsync(CancellationToken ct = default);
}
```

**SaveAsync logic:**
1. Serialize snapshot (with ContentHash=null) using `MessagePackSerializer.Serialize` with `ContractlessStandardResolver`.
2. Compute XxHash128 over the serialized bytes. Convert to hex string.
3. Create new snapshot with ContentHash set: `snapshot with { ContentHash = hashHex }`.
4. Re-serialize (now with ContentHash) — this is the final bytes written to disk.
   Actually, the content hash should be over the content WITHOUT the hash itself to avoid circular dependency. So: hash the bytes with ContentHash=null, then write the snapshot with ContentHash set. The file bytes will differ from what was hashed, but the hash identifies the content.
5. Write to `{artifactsDir}/{hashHex}.msgpack`.
6. Update manifest.json (see below).

**Manifest schema (per research recommendation):**
```json
{
  "version": "1.0",
  "snapshots": [
    {
      "contentHash": "abc123...",
      "projectName": "MyProject",
      "gitCommitSha": "def456...",
      "ingestedAt": "2026-02-26T12:00:00Z",
      "schemaVersion": "1.0",
      "nodeCount": 150,
      "edgeCount": 300
    }
  ]
}
```

Use `System.Text.Json` for manifest (in-box, no extra dependency).

**Manifest update:** Read existing manifest.json if present, add new entry, write back. If file doesn't exist, create new. Use file locking or atomic write (write to temp, rename) to avoid corruption.

**LoadAsync:** Read `{artifactsDir}/{contentHash}.msgpack`, deserialize with MessagePack + ContractlessStandardResolver.

**ListAsync:** Read manifest.json, deserialize, return entries.

**Retention policy (Claude's discretion):** Keep all snapshots for V1. No pruning logic needed.
