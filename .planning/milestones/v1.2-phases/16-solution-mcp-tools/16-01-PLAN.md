<task type="auto">
  <name>Task 1: Create SolutionTools class with explain_solution tool</name>
  <files>src/DocAgent.McpServer/Tools/SolutionTools.cs</files>
  <action>
Create `SolutionTools.cs` in `src/DocAgent.McpServer/Tools/` following the exact ChangeTools pattern:

1. **Class structure:** `[McpServerToolType] public sealed class SolutionTools` with constructor taking `SnapshotStore`, `PathAllowlist`, `ILogger<SolutionTools>`, `IOptions<DocAgentServerOptions>`.

2. **Static JsonSerializerOptions:** Same as ChangeTools — `WriteIndented = false`, `PropertyNamingPolicy = JsonNamingPolicy.CamelCase`.

3. **explain_solution method:**
   - Signature: `[McpServerTool(Name = "explain_solution")] [Description("...")]` returning `Task<string>`
   - Parameters: `string snapshotHash` (content hash from ingest_solution), `CancellationToken`
   - PathAllowlist guard first: `if (!_allowlist.IsAllowed(_snapshotStore.ArtifactsDir))` → return opaque "Solution not found." error
   - Load snapshot via `_snapshotStore.LoadAsync(snapshotHash, ct)` — if null, return "Solution not found." error
   - Check if solution snapshot: if `snapshot.SolutionName` is null, treat as single-project; if all Real nodes have same/null ProjectOrigin, set `isSingleProject = true`
   - **Per-project stats:** Group `NodeKind.Real` nodes by `ProjectOrigin`. For each project compute:
     - `nodeCount`: count of Real nodes
     - `edgeCount`: count of IntraProject edges where From or To is in that project's node set
     - `docCoveragePercent`: % of public/protected/protectedInternal nodes of kinds {Type, Method, Property, Constructor, Delegate, Event, Field} that have non-null `Docs?.Summary`. Round to 1 decimal.
   - **Dependency DAG:** From `EdgeScope.CrossProject` edges, build nodeId→projectName lookup, collect unique `(fromProject, toProject)` pairs, output as adjacency list dict `{ "ProjectA": ["ProjectB", "ProjectC"] }`
   - **Stub count:** Count all nodes where `NodeKind == NodeKind.Stub`
   - **JSON response shape** (per CONTEXT.md locked decision):
     ```json
     {
       "solutionName": "...",
       "snapshotId": "hash",
       "projects": [ { "name": "...", "nodeCount": N, "edgeCount": N, "docCoveragePercent": N.N } ],
       "dependencyDag": { "A": ["B"] },
       "totalStubNodeCount": N,
       "isSingleProject": false
     }
     ```
   - Serialize with `JsonSerializer.Serialize(result, s_jsonOptions)` and return.

4. **Helper methods:** Private `ErrorResponse` method returning JSON error string matching ChangeTools pattern. Private method for doc coverage computation.

Use `System.Text.Json` throughout. Do NOT reference `SolutionSnapshot` or `SolutionIngestionResult` at runtime — work entirely from the flat `SymbolGraphSnapshot`.
