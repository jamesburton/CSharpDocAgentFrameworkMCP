<task type="auto">
  <name>Task 1: Add diff_snapshots tool to SolutionTools</name>
  <files>src/DocAgent.McpServer/Tools/SolutionTools.cs</files>
  <action>
Add `diff_snapshots` method to the existing `SolutionTools` class:

1. **Method signature:**
   ```csharp
   [McpServerTool(Name = "diff_snapshots")]
   [Description("Diff two solution snapshots showing per-project changes, projects added/removed, and cross-project edge changes.")]
   public async Task<string> DiffSnapshots(
       [Description("Before snapshot version (content hash)")] string before,
       [Description("After snapshot version (content hash)")] string after,
       CancellationToken cancellationToken = default)
   ```

2. **PathAllowlist guard** — same pattern as explain_solution.

3. **Load both snapshots** via `_snapshotStore.LoadAsync()`. If either is null, return "Solution not found." error.

4. **Group nodes by ProjectOrigin** for both snapshots (filter to `NodeKind.Real` only):
   - `beforeProjects`: Dictionary<string, List<SymbolNode>> from before snapshot
   - `afterProjects`: same from after snapshot

5. **Detect projects added/removed:**
   - `projectsAdded`: project names in after but not before
   - `projectsRemoved`: project names in before but not after
   - `survivingProjects`: project names in both

6. **Per-project diffs for surviving projects:**
   - For each surviving project, reconstruct a per-project `SymbolGraphSnapshot` from the flat snapshot using a private helper `ExtractProjectSnapshot(SymbolGraphSnapshot flat, string projectName, IReadOnlyList<SymbolNode> nodes)`:
     - Set `ProjectName = projectName`
     - Filter edges to `IntraProject` scope where From or To is in the project's node ID set
     - Use `flat with { ProjectName = projectName, Nodes = [...], Edges = [...] }`
   - Call `SymbolGraphDiffer.Diff(beforeProjectSnapshot, afterProjectSnapshot)`
   - Collect results as `{ "projectName": { "added": N, "removed": N, "modified": N, "changes": [...] } }`

7. **Cross-project edge diff:**
   - Collect all `CrossProject` edges from before and after snapshots
   - Compute added = in after but not before, removed = in before but not after
   - Compare edges by (From.Value, To.Value, Kind) tuple
   - For each added/removed edge, include `from` (with project name), `to` (with project name), `kind`

8. **JSON response shape** (per CONTEXT.md locked decision):
   ```json
   {
     "before": "hash-before",
     "after": "hash-after",
     "projectsAdded": ["NewProject"],
     "projectsRemoved": [],
     "projectDiffs": {
       "DocAgent.Core": { "added": 2, "removed": 0, "modified": 3, "changes": [...] }
     },
     "crossProjectEdgeChanges": {
       "added": [{ "from": "Project::Symbol", "to": "Project::Symbol", "kind": "References" }],
       "removed": []
     }
   }
   ```

9. **Edge equality:** Compare edges by `(From.Value, To.Value, Kind)` tuple. Create a private record or use anonymous type with proper equality for HashSet operations.
