**Core IngestAsync logic (sequential project processing — MSBuildWorkspace is NOT thread-safe):**

1. Open solution: `MSBuildWorkspace.Create()` + `OpenSolutionAsync(slnPath)`. Capture `WorkspaceFailed` events into warnings list. If `OpenSolutionAsync` throws, return a result with zero projects and the error as a warning.

2. Group projects by `FilePath` (case-insensitive) to detect multi-targeted duplicates. For each group with >1 entry, pick the one with the highest TFM using `ExtractTfmVersion` (parse from project name pattern `"Name (netX.Y)"`; modern `netX.Y` > legacy `netNN`; fallback to Version(0,0) if no pattern match). Record chosen TFM in status.

3. For each chosen project entry:
   - Filter out `project.FilePath is null` → status "skipped", reason "No file path"
   - Filter out non-C# via `project.Language != LanguageNames.CSharp` → status "skipped", reason `$"Unsupported language: {project.Language}"`
   - Call `project.GetCompilationAsync(ct)`. If null → status "failed", reason "Could not obtain compilation"
   - If compilation succeeds: reuse the node/edge walking logic. Use `RoslynSymbolGraphBuilder` internally (construct one with `XmlDocParser` + `InheritDocResolver`) OR extract the walk logic. Simplest approach: create a new `RoslynSymbolGraphBuilder`, call its existing `BuildAsync` equivalent on the single project, then stamp `ProjectOrigin` on all returned nodes via `with` expression: `n with { ProjectOrigin = projectName }`. The project name is `Path.GetFileNameWithoutExtension(project.FilePath)` (the clean project name, not the TFM-suffixed Roslyn name).
   - Add nodes and edges to running lists. Record status "ok" with nodeCount and chosenTfm.

4. Build ProjectEntry list and ProjectEdge DAG from `solution.Projects` — include ALL projects (even non-C#) in the DAG per CONTEXT.md decision "Preserve cross-language dependency edges". Use `project.AllProjectReferences` to build edges.

5. Merge all nodes/edges into a single flat `SymbolGraphSnapshot`:
   - SchemaVersion: "1.2"
   - ProjectName: solutionName (the .sln filename without extension)
   - SolutionName: solutionName
   - SourceFingerprint: hash of slnPath + timestamp (or content hash)
   - Nodes: all collected nodes (already stamped with ProjectOrigin)
   - Edges: all collected edges

6. Persist via `_store.SaveAsync(snapshot, ct)` and index via `_index.IndexAsync(saved, ct)` (soft failure on index, same as IngestionService).

7. Return `SolutionIngestionResult` with all statuses, warnings, snapshot ID.

**TFM version extraction helper:**
```csharp
private static Version ExtractTfmVersion(string projectName) {
    var match = Regex.Match(projectName, @"\(net(\d+)\.(\d+)\)$");
    if (match.Success && int.TryParse(match.Groups[1].Value, out var major)
        && int.TryParse(match.Groups[2].Value, out var minor))
        return new Version(major, minor);
    var legacy = Regex.Match(projectName, @"\(net(\d+)\)$");
    if (legacy.Success && int.TryParse(legacy.Groups[1].Value, out var lv))
        return new Version(0, lv);
    return new Version(0, 0);
}
```
