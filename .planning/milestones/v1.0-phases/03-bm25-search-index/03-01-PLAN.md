**Step 2: Create `CamelCaseAnalyzer.cs`.**

A sealed `Analyzer` subclass using:
- `WhitespaceTokenizer` as source
- `WordDelimiterFilter` with flags: `SPLIT_ON_CASE_CHANGE | GENERATE_WORD_PARTS | GENERATE_NUMBER_PARTS | PRESERVE_ORIGINAL`
- `LowerCaseFilter` at the end

This produces tokens like: `GetReferences` → `[getreferences, get, references]`, `XMLParser` → `[xmlparser, xml, parser]`.

Use `LuceneVersion.LUCENE_48` throughout.

**Step 3: Create `BM25SearchIndex.cs` implementing `ISearchIndex`.**

Key design:
- Constructor takes `string artifactsDir` (path for persistent index storage). Also accept an optional `Lucene.Net.Store.Directory` parameter for testing with `RAMDirectory`.
- Internal `Dictionary<SymbolId, SymbolNode> _nodes` for `GetAsync` O(1) lookup (same pattern as `InMemorySearchIndex`).
- `PerFieldAnalyzerWrapper`: `CamelCaseAnalyzer` for `symbolName` and `fullyQualifiedName` fields, `StandardAnalyzer` for `docText` field.
- Custom `PerFieldSimilarityWrapper` subclass (`DocAgentSimilarity`): `BM25Similarity(k1: 2.0f, b: 0.5f)` for `symbolName`, default `BM25Similarity()` for other fields.

**IndexAsync:**
- Populate `_nodes` dictionary from snapshot.
- Open Lucene directory (FSDirectory or injected).
- Create `IndexWriterConfig` with `OpenMode.CREATE`, the per-field analyzer, and `DocAgentSimilarity`.
- For each `SymbolNode` in `snapshot.Nodes`, add a Document with fields:
  - `symbolId` (StringField, Store.YES) — `node.Id.Value`
  - `symbolName` (TextField, Store.YES) — `node.DisplayName ?? ""`
  - `fullyQualifiedName` (TextField, Store.NO) — `node.FullyQualifiedName ?? ""`
  - `docText` (TextField, Store.NO) — `node.Docs?.Summary ?? ""`
- Commit and dispose writer.

**SearchAsync:**
- If no index exists, yield break.
- Open `DirectoryReader`, create `IndexSearcher` with `DocAgentSimilarity`.
- Tokenize the query string through `CamelCaseAnalyzer.GetTokenStream()` to get individual tokens.
- Build a `BooleanQuery` with `Occur.SHOULD` clauses for each token across `symbolName` and `docText` fields.
- Search top 50, yield `SearchHit` for each result.

**GetAsync:**
- Return from `_nodes` dictionary by `SymbolId`.

**IDisposable:** Implement `IDisposable` to dispose the Lucene directory if owned.
