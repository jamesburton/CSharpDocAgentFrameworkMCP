**RoslynSymbolGraphBuilder** implements `ISymbolGraphBuilder`. Constructor takes `XmlDocParser` and `InheritDocResolver` as dependencies.

**BuildAsync implementation:**
1. Open solution/projects using MSBuildWorkspace (reuse patterns from LocalProjectSource or accept pre-loaded workspace).
   - Since the interface takes `ProjectInventory` + `DocInputSet`, iterate `inv.ProjectFiles`.
   - For each project, open via MSBuildWorkspace, get `Compilation`.
   - **Important:** Process one project at a time. Null references to Compilation after processing to release memory (per research pitfall 4).

2. **Recursive symbol walker** — for each Compilation:
   ```
   WalkNamespace(compilation.GlobalNamespace, nodes, edges)
   ```

   **WalkNamespace(INamespaceSymbol ns):**
   - For each member (ordered by `GetDocumentationCommentId()`):
     - If `INamespaceSymbol` → recurse, add Contains edge
     - If `INamedTypeSymbol` → check accessibility (public/protected only), add node + Contains edge, call WalkType

   **WalkType(INamedTypeSymbol type):**
   - Add inheritance edges: `type.BaseType` → SymbolEdgeKind.Inherits
   - Add interface implementation edges: `type.Interfaces` → SymbolEdgeKind.Implements
   - Walk nested types recursively (full depth per user decision)
   - Walk members: methods, properties, fields, events, constructors, operators, indexers
   - For each member: check accessibility, create SymbolNode, add Contains edge

   **For each member, also add:**
   - Parameter/return type reference edges (SymbolEdgeKind.References)
   - Override edges for overridden methods (SymbolEdgeKind.Overrides)

3. **Symbol to SymbolNode mapping:**
   - `Id`: use `ISymbol.GetDocumentationCommentId()` as SymbolId value. If null (rare), use `ISymbol.ToDisplayString()`.
   - `Kind`: map `ISymbol.Kind` / `MethodKind` to project's `SymbolKind` enum
   - `DisplayName`: `ISymbol.Name`
   - `FullyQualifiedName`: `ISymbol.ToDisplayString()`
   - `PreviousIds`: empty list for V1 (rename tracking is future)
   - `Accessibility`: map `ISymbol.DeclaredAccessibility` to project's `Accessibility` enum
   - `Docs`: call `XmlDocParser.Parse(symbol.GetDocumentationCommentXml())`. If null, use InheritDocResolver. If still null, synthesize placeholder: `new DocComment("No documentation provided", null, empty, empty, null, empty, empty, empty)`.
   - `Span`: extract from `symbol.Locations.FirstOrDefault()` where `IsInSource` is true, using `GetLineSpan()`. Convert 0-based to 1-based line/column numbers.

4. **Generated code detection (per user decision):**
   - Check `[GeneratedCode]` attribute via `symbol.GetAttributes()`
   - Check source file path for `/obj/` or `\obj\`
   - Tag but include — add a metadata flag. Since SymbolNode doesn't have a flag field, consider adding `IsGenerated` info into the DisplayName suffix or as a convention in the SymbolId (e.g., prefix). OR add the flag: if adding a field to SymbolNode is needed, add `bool IsGenerated` to the record. Check if this requires updating Core — if so, add it.

5. **Construct SymbolGraphSnapshot:**
   - SchemaVersion: "1.0"
   - ProjectName: from project name
   - SourceFingerprint: compute from sorted project file hashes
   - ContentHash: null (set by SnapshotStore in Plan 04)
   - CreatedAt: `DateTimeOffset.UtcNow`
   - Nodes/Edges: sorted by SymbolSorter

**SymbolSorter:**
```csharp
public static class SymbolSorter
{
    public static IReadOnlyList<SymbolNode> SortNodes(IEnumerable<SymbolNode> nodes)
        => nodes.OrderBy(n => n.Id.Value, StringComparer.Ordinal).ToList();

    public static IReadOnlyList<SymbolEdge> SortEdges(IEnumerable<SymbolEdge> edges)
        => edges
            .OrderBy(e => e.From.Value, StringComparer.Ordinal)
            .ThenBy(e => e.To.Value, StringComparer.Ordinal)
            .ThenBy(e => e.Kind)
            .ToList();
}
```
