  <action>
Create `tests/DocAgent.Tests/SnapshotSerializationTests.cs` with these tests. Add `using MessagePack;` and `using System.IO.Hashing;`.

**Setup helper:** Create a private method that builds a representative SymbolGraphSnapshot with at least 2 nodes, 1 edge, a DocComment with all fields populated (Params, TypeParams, Exceptions, SeeAlso, Examples), PreviousIds on one node, and all enum values exercised. Use `ContentHash: null` initially.

**MessagePack serializer options:** Use `MessagePackSerializerOptions.Standard.WithResolver(MessagePack.Resolvers.ContractlessStandardResolver.Instance)` for all serialize/deserialize calls. Store as a static field.

**Tests:**

1. `Roundtrip_MessagePack_produces_identical_snapshot` — serialize snapshot to byte[], deserialize back, assert all fields match using FluentAssertions `.BeEquivalentTo()` (handles dictionary equality). This proves the MessagePack contract works with ContractlessStandardResolver on record types.

2. `Serialization_is_deterministic` — serialize the same snapshot twice (two separate calls), assert byte arrays are identical (`SequenceEqual`). This proves determinism required by CORE-02.

3. `ContentHash_is_stable_across_serializations` — serialize snapshot to bytes, compute XxHash64 hash:
   ```csharp
   byte[] bytes = MessagePackSerializer.Serialize(snapshot, options);
   ulong hash = XxHash64.HashToUInt64(bytes);
   string contentHash = hash.ToString("x16");
   ```
   Do this twice. Assert both hash strings are identical. Then set `snapshot with { ContentHash = contentHash }` and verify the field is set.

4. `Json_roundtrip_produces_equivalent_snapshot` — serialize with `System.Text.Json.JsonSerializer` using `JsonSerializerOptions { PropertyNamingPolicy = JsonNamingPolicy.CamelCase, WriteIndented = true }`. Deserialize back. Assert `.BeEquivalentTo()`. This proves JSON debugging format works.

5. `ContentHash_differs_for_different_snapshots` — create two snapshots with different nodes. Compute ContentHash for each. Assert they differ.

**Note on System.IO.Hashing:** `XxHash64` is in-box for .NET 10 — no additional package reference needed.
