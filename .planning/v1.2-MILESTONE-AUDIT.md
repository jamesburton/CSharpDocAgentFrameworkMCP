### Critical Defect: Duplicate MCP Tool Name

**DEFECT-01:** `diff_snapshots` registered in both:
- `src/DocAgent.McpServer/Tools/DocTools.cs:375` — flat per-symbol diff (Phase 5, MCPS-04)
- `src/DocAgent.McpServer/Tools/SolutionTools.cs:164` — solution-level per-project diff (Phase 16, TOOLS-04)

Both use `[McpServerTool(Name = "diff_snapshots")]`. MCP protocol requires unique tool names. `WithToolsFromAssembly()` discovers both at startup.

**Fix:** Rename `SolutionTools.DiffSnapshots` to `diff_solution_snapshots` (or rename the DocTools version and update existing consumers).
